# EV Assist

Aplikacja webowa do kalkulacji oszczędności przy zakupie samochodu elektrycznego.

## Funkcjonalności

- Kalkulacja oszczędności na podstawie przejechanych kilometrów
- Porównanie kosztów paliwa vs prąd
- Obsługa wielu języków (PL/EN)
- Responsywny design

## Technologie

- React + TypeScript
- Vite
- ESLint
